# ZXingTest
使用ZXing來產生QR Code和掃描QR Code
並且可以將QRCode分享到FB,LINE,Email,Whatsapp的測試專案

## install cocoaPods ZXingObjC

```ruby
platform :ios, '7.0'
pod 'ZXingObjC', '~> 3.0'
```
